# Flare UI

Frontend for the [Flare](https://github.com/flrapp) feature flag management system.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env`:
   ```env
   VITE_API_BASE_URL=http://localhost:5000
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```
